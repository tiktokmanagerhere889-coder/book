# Chatbot Integration

The Book Assistant chatbot is integrated throughout the entire documentation site, providing instant access to information about the book content on every page.

## Features

- **Persistent Access**: The chatbot is available on every page of the book
- **Knowledge Base**: Answers questions based on the book content and related materials
- **Citations**: Provides source citations for all answers
- **Confidence Scores**: Shows confidence levels for each response
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Accessibility**: Fully accessible with keyboard navigation and screen reader support

3. Press Enter or click the send button to submit your question
4. Review the response along with citations and confidence scores
5. Minimize the chat by clicking the minimize button when you're done

## Keyboard Shortcuts

- `Ctrl/Cmd + Shift + K`: Open or close the chat interface

## Privacy

The chatbot does not store personal information. Conversations are stored locally in your browser for session continuity but are not transmitted to any external services except to process your current query.

## Troubleshooting

If you encounter issues with the chatbot:

- Check your internet connection
- Refresh the page if the chatbot becomes unresponsive
- Clear your browser's local storage if conversation history appears corrupted
